Enterprise Secure Access Gateway & Zero Trust Lab

onepercentplaid.com

Engineered a multi-tier security architecture to host and protect sensitive Windows Active Directory services within a "Dark Server" environment. This project demonstrates a transition from traditional perimeter security to a modern, identity-centric **Zero Trust model**, ensuring critical infrastructure remains invisible to the public internet.

  • Zero-Trust Mesh Networking (WireGuard/Tailscale)

    Eliminated the need for legacy VPNs and port forwarding by deploying an encrypted WireGuard-based mesh. This architecture ensures the Domain Controller has no public IP presence and is reachable only via authenticated nodes.

  • Reverse Proxy & Identity Gatekeeping

    Deployed a hardened Nginx instance for SSL termination and traffic orchestration, integrated with Duo MFA to provide a mandatory secondary authentication layer for the Apache Guacamole portal.

  • Active Defense & Log Parsing (Fail2Ban)

    Developed custom 'jail' configurations to monitor Nginx and system logs. The system automatically executes firewall-level IP bans upon detection of brute-force attempts or unauthorized directory traversal.

  • Forensic Auditing & Compliance

    Implemented clientless RDP/SSH access with full video session recording. This creates a tamper-proof audit trail for administrative actions, essential for security compliance and incident response.

  • Python-Driven Security Telemetry

    Authored an autonomous Python watchdog service running as a systemd unit. It monitors critical service health and log anomalies, pushing real-time incident reports to a Discord-based SOC via webhooks.

Self-Hosted Remote Support Infrastructure

Private Remote Relay Deployment

Architected and deployed a private, self-hosted RustDesk infrastructure on **Windows Server 2025** to eliminate reliance on third-party SaaS providers. This project focused on maintaining absolute **Data Sovereignty** and implementing a "CLI-Over-GUI" philosophy for reliable, automated systems management.

  • Automated Service Availability

    Engineered a PowerShell-driven automation workflow to register the RustDesk ID (hbbs) and Relay (hbbr) services as system-level background tasks. This ensures 24/7 availability and automatic recovery upon system reboot without requiring manual user intervention.

  • Identity Management & Least Privilege

    Integrated with **Active Directory Domain Services (AD DS)** to manage technician access. Implemented group nesting (RemoteSupportUsers into Account Operators) to adhere to the Principle of Least Privilege while ensuring necessary administrative permissions for remote sessions.

  • Cryptographic Hardening & E2EE

    Secured all remote sessions using the **NaCL Cryptographic Library**. Deployed ed25519 public key authentication to mitigate Man-in-the-Middle (MITM) attacks and disabled public relaying to ensure all session metadata remains within the internal network perimeter.

  • PowerShell-First Configuration

    Bypassed GUI limitations by using PowerShell for advanced infrastructure tasks, including ICMP traffic enablement for network discovery, custom firewall rule creation (TCP 21115-21119), and modifying Task Scheduler power restrictions.
