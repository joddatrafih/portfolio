[01] Enterprise Secure Access Gateway & Zero Trust Lab

onepercentplaid.com

Engineered a hardened "Dark Server" infrastructure to isolate and protect a Windows Active Directory Domain Controller. Developed a multi-layered security stack that transitions from passive isolation to active intrusion defense.

  • Zero-Trust Networking (ZTNA) Leveraged Tailscale Mesh VPN and Nginx Reverse Proxy to eliminate public-facing attack surfaces, ensuring resources are invisible to the open internet.
  • Active Intrusion Prevention Configured Fail2Ban with custom regex filters to monitor Nginx access logs, automatically triggering firewall-level IP bans (iptables) upon detection of brute-force attempts.
  • Forensic Auditing Integrated an automated Session Recording system that generates tamper-proof video audits of every RDP/SSH connection, mapped via UUID for administrative playback.
  • Autonomous Alerting Developed a custom Python Watchdog service running as a systemd unit to provide real-time security telemetry and incident notifications via Discord Webhooks.
